# 🧮 Java CLI Calculator

A lightweight command-line calculator built in Java that performs basic arithmetic operations using user input and clean conditional logic.

---

## ⚡ Overview

This project allows users to input two numbers and choose an operation to calculate the result instantly via terminal interaction.

---

## 🛠️ Features

- Supports operations: `+`, `-`, `*`, `/`, `%`
- Interactive CLI input
- Fast and simple execution
- Formatted output with precision

---

## 🧩 Tech Stack

- Java  
- Scanner (user input)  
- Ternary operators  
- Basic control flow  

---
